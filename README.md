# Sunflower — двухосевой трекер света на STM32F103

## 🔍 Общая концепция
Устройство на базе отладочной платы **Blue Pill Plus (STM32F103C8T6)** управляет двухосевым поворотным механизмом с двумя сервоприводами. 
Алгоритм:
1. **Режим разведки (Scan mode)** — сервоприводы поочередно обходят заданные сектора, измеряя освещенность в каждой точке.
2. **Анализ** — нахождение углов с максимальным значением с датчиков света.
3. **Режим слежения (Tracking mode)** — удержание источника света при его перемещении.

## 🧠 Алгоритм работы
- **Инициализация** — датчики света (фоторезисторы) подключены к АЦП, сервы — к ШИМ-выходам таймеров, LCD-дисплей для вывода показаний освещенности (I2C).
- **Первичный поиск максимума** — сканирование по азимуту (0–180°) с шагом 10° и меридиану (0–180°) с шагом 20°. Сохраняется пара углов с наибольшим уровнем сигнала.
- **Слежение** — движение за источником света с наибольшей освещенностью. 
- **Точность** — для более расширенной работы есть возможность изменять чувствительность прибора.

## ⚙️ Используемые компоненты
- **MCU:** STM32F103C8T6 (Blue Pill Plus)
- **Приводы:** 2 x сервопривод SG90
- **Сенсоры:** 2 x фоторезистор 
- **Питание:** 5В / 3.3В 

## 📁 Структура кода
- `main.c` — основная логика, конечный автомат (SCAN → ANALYZE → TRACK)
- `lcd1602_i2c.c` - драйвер дисплея
- `servo.c` — драйвер сервоприводов 
- `adc.c` — опрос датчиков света
- `lcd1602_i2c.h` - обозначение регистров
- `servo.h` —  преобразования угла в импульс
- `adc.h` — преобразование показаний датчиков в %
