# Sunflower
## 🔍 Общая концепция
Устройство на базе отладочной платы Blue Pill Plus (STM32F103C8T6) управляет двухосевым поворотным механизмом с двумя сервоприводами.
Алгоритм:
1) Режим разведки — сервоприводы поочередно обходят заданные сектора, измеряя освещенность в каждой точке;
2) Анализ — нахождение углов с максимальным значением с датчиков света;
3) Режим слежения — удержание источника света при его перемещении.
## 🧠 Алгоритм работы
- **Инициализация** — датчики света (фоторезисторы/фотодиоды) подключены к АЦП, сервы — к ШИМ-выходам таймеров.
- **Первичный поиск максимума** — сканирование по азимуту (0–180°) и углу места (0–90°) с шагом, например, 15°. Сохраняется пара углов `(pan, tilt)` с наибольшим уровнем сигнала.
- **Слежение** — малые периодические колебания (dithering) вокруг найденного положения и корректировка, если соседняя область дает больше света.
- **Точность** — при необходимости используется интерполяция (парабола по 3 точкам) для более точного наведения.

## ⚙️ Используемые компоненты
- **MCU:** STM32F103C8T6 (Blue Pill Plus)
- **Приводы:** 2 x сервопривод (SG90 / MG995 / аналоги)
- **Сенсоры:** 2 x фоторезистор или 2 x фотодиод (с делителем напряжения)
- **Питание:** 5В / 3.3В (через стабилизаторы платы)

## 📁 Структура кода (пример)
- `main.c` — основная логика, конечный автомат (IDLE → SCAN → ANALYZE → TRACK)
- `servo.c` — драйвер сервоприводов (PWM, углы)
- `adc.c` — опрос датчиков света
- `tracking_alg.c` — поиск экстремума, слежение
- `scan_patterns.h` — таблицы углов для сканирования
